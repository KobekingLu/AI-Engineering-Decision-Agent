# v0.1.0

Initial public release of the AI Engineering Decision Agent.

## What Is Included

- decision-agent core for single-case intake, retrieval, classification, and summary generation
- local Streamlit demo app for pasted text and multi-file case bundles
- benchmark regression flow for sample decision cases
- OCR support for scanned PDFs and images when available in the environment
- governed knowledge-positioning documentation for uncertainty-aware decision assistance

## Highlights

- stage-aware decision assistance with lifecycle-sensitive inference
- issue family and subtype classification
- missing-information and next-step guidance
- single-case JSON and HTML report output
- basic GitHub Actions validation for syntax and smoke-case execution

## Validation

- `python -m compileall decision_agent web_app`
- `python -m decision_agent.run_case decision_agent/input/case_001_normalized.json`

## Notes

- This is a demo-grade release focused on decision assistance under incomplete information.
- The project is not intended to memorize closed benchmark answers or replace human engineering sign-off.
